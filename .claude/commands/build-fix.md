ビルドエラーや型エラーを段階的に修正する。

## 手順

1. ビルドを実行: `npm run build` または該当コマンド
2. エラー出力を解析:
   - ファイルごとにグループ化
   - 重要度順にソート
3. 各エラーに対して:
   - エラーの前後5行のコンテキストを表示
   - 問題の原因を説明
   - 修正を提案・適用
   - ビルドを再実行
   - エラーが解消されたことを確認
4. 以下の場合は停止:
   - 修正が新しいエラーを生んだ場合
   - 同じエラーが3回の試行後も解消しない場合
5. サマリーを表示:
   - 修正したエラー数
   - 残存エラー数
   - 新規に発生したエラー数

## 重要

- 一度に1つのエラーを修正する（安全のため）
- 修正ごとにビルドを再実行して確認する
- 新しいエラーが発生した場合はロールバックを検討する
