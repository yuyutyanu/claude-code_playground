# .claude/

このディレクトリには、AIアシスタントが参照する技術ドキュメントとベストプラクティスガイドが含まれています。

## ディレクトリ構造

```
.claude/
├── README.md           # このファイル
└── skills/             # 技術スタック別のベストプラクティス
    ├── nextjs.md       # Next.jsの開発ガイド
    ├── vitest.md       # Vitestのテストガイド
    └── biome.md        # Biomeの設定とフォーマットガイド
```

## 使用方法

各スキルドキュメントには、特定の技術に関するベストプラクティス、コーディング規約、よくある問題の解決方法が記載されています。

AIアシスタントはこれらのドキュメントを参照して、一貫性のある高品質なコードを生成します。

## ドキュメントの更新

プロジェクトの成長に応じて、以下を追加・更新してください：
- 新しい技術スタックのガイド
- プロジェクト固有のパターンと規約
- よくある問題とその解決方法
- パフォーマンス最適化のヒント
